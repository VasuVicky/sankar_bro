Now i want to add an image to entire the table and for each row and each column or colors.


To add image to the table in 3 ways :


1.Using background-image attribute with image path as value with single inverted comas (mandatory) in the table tag it applies the image to the entire table., and set the width and height for adjustments.

2.Using inline styles : style attribute with property background-image with value as url(image path without using double quotes).

3. Using Internal styles using style tag inside the head part with property background-image : url(); and to set the image compatable to the row or column using background-size property with value cover like background-size : cover ;
4. we can also insert the image using img tag inside the td by adjusting width and height of the image.. using background property in style attribute in 2 ways : background : color or background : url(image). every property and value in style attribute terminates with semicolon ;
To move the image from left - right, top - bottom by using css property position with value relative and to adjust using left , right , top , bottom with no.of px values without position property l,r,top , btm are not worked.

The properties of the css are same in all type of css like inline (embeded) , internal with in the style tag or in external css .
The way of style organization is different but the properties and values are same .
If you want to use the styles within the tag then using : inline css (not better choice it decrease the code readability.)
If you want to use the style in head part also using style tag by implementing selectors with property and value(which is the best way to divide the body called structure designing and styles).
If you want to increase the code readability and performance to by separating the structure designing and style using External css(which is highly recomended).


we cannot adjust the height and width of the image existed in entire row using background-image property , but we can able to do this using img tag with in a particular cell called : td / th.

Keep in mind , don't make any mistake , keep the properties and value with in the attribute and keep attributes with in the tag. otherwise it will display as a normal content on the page.
