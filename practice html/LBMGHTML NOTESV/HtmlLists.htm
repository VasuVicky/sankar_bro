List is a collection of Items.
List is widely used tag in html to arrange the values in a order / unorderd manner.
For example list of items in a menu card on the web for hotels, a list of courses in colleges etc.


There are 3 type of lists in html :


Orderd List : Arranging the group of elements in a sequntial order either in alphabetical in small or capital or in roman number either in capitial or small or numerics etc.
To define the order list using : ol tag. and it is a collection of items called list Items .
To define the list items using li tag.
Ol tag expects one attribute called type attribute and holds value of the type of the items sequence mentioned in above lines.
And we can also specify the starting range , from which value it will be started. using start attribute with value : start range , if you want to start the sequence from 5 then you can start = "5".
Default value of the orderd lis is numrics.
