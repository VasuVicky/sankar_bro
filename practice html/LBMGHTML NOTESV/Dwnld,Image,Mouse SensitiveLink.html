Mouse Sensitive Links : HTML & XHTML provides the feature of embeding multiple links in a single image.
1.We can create a different multiple link in a single image based on the different coordinates on the image.
2.Then we can select & click the desired target link from the different coordinates in the single imange.
3.Mouse sensitive links in a single image on different coordinates are also called Image Maps.
Types :
These are two types :
1.Server Side Image Maps . 2.Client Side Image Maps.


Every Image map have an attribute :
1. If you want to use Server Side Image Map by using : ismap attribute. 2. If you want to use Client Side Image Map by using : usemap attribute.
1.Server Side Image Maps : ismap.
The server use the Mouse Pointer coordinates for which document to be deliver back to the browser.
keep use the ismap attribute in img tag. it works like a normal link means redirecting to the server. like

2.Client Side Image Maps : usemap.
The client use the Mouse Pointer coordinates for which document to be deliver back to the browser.
keep use the usemap attribute in img tag by passing a value with #tag : #value and keep maintain this value as name attribute value in map tag. like
When we implementing the usemap we can use two tags called , map tag and area tag.
map tag expects an attribute called name : and give any name as value , keep equal the value of the name attribute and usemap attribute value in img tag.
area is the another tag inside the map tag , we can create n no. of area tags inside the map tag, and it expects 3 attributes : shape="circle" and coords="60,60,30,10" : coordinate points. and href ="link".
