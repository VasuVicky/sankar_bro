Layouts : A layout is the area of the window , which specifies the subdivisons of the elements, a layout provides a pattern where the element need to placed, layouts plays morst important role in the webpage , without layout we can't add our elements correctly.Throught layouts only we add the desired html elements to our page which defines the designing of the page. Layouting : The process of placed/arranging the html elements on desired locations based on our requirements,
Every application is developed through layout only with out layout there is no application structure, To define a Layouts using flexbox : properties.
Flexbox is the principle to implements the layout of the window
and There are a few list of properties to design layout are :
Methods to design a layout :
  1. Flexbox Container
  2. Flexbox Direction
  3. Justify content
  4. CSS GRID : it is not best way / not stable to develop layout we are learn later after it get stable now we learn about flexbox to develope layouts

1.Flexbox Container : To design a layout then first we need to descide which type container we will use. that container is called flex container. Flex container is a collection of flex items, all html element inside the flex box container called flex items.

2.Flex Direction : flex direction is the property specifies the way of arranging the flex items either in vertically /column wise or horizontally /row wise
3.Justify Content : it is the class property of the bootstrap layouts which specifies the alignment of the flex items based on the flex direction.
If the flex direction is horizontal then justify content : left(start) or center or right(end).
If the flex direction is vertical then the justify content : top(strt) or center or bottom(end) : which specifies the position of the flex item where it will be displaying.

Steps to create the flex content :
  • Create a flexbox container
  • Add elements or flex items in it
  • specifies the flex direction
  • justify the content based on your requirements.
    • To define the layouts boostrap is the best choice to reuse the predefined classes to arrange the layouts.

1.Create a background container as a flex box container using predefined bootstrap class called "d-flex" (display flex).

2.Specify the direction of the flex item/items horizontally by using class called : "flex-row" / vertically : "flex-column".

3.Justify the content to the center or end or start based on your needs. to align the flex items to the center position using align-items-center bootrap class name
